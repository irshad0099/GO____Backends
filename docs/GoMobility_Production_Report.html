3. AWS Infrastructure Plan (Production Migration)

🏗 Recommended AWS Architecture

Internet → Route 53 (api.gomobility.co.in)
    → CloudFront (CDN + SSL termination)
    → ALB (Application Load Balancer)
        → EC2 t3.medium [Node.js #1] (AZ-a)
        → EC2 t3.medium [Node.js #2] (AZ-b) [Scale-out]
        → RDS PostgreSQL db.t3.small (Multi-AZ optional)
        → ElastiCache Redis t3.micro (cluster mode)
        → S3: go-mobility-documents (KYC uploads)
        → S3: go-mobility-static (profile photos)
VPC → Private Subnet (DB + Cache) | Public Subnet (EC2 + ALB)
Security Groups → EC2 allows only ALB traffic; RDS allows only EC2
