Ride Flow — Event Timeline

Complete Ride Lifecycle (Socket Events)

1
Passenger books ride → Server emits ride:new_request to nearby drivers
2
Driver accepts → Server emits rideAccepted to passenger + driver
3
Driver moving to pickup → Driver emits driver:location_update every 5s → Server emits driverLocationUpdated to ride room
4
Driver reaches pickup → Server emits driverArrived to passenger + driver (with OTP)
5
Passenger shares OTP → Driver verifies → Server emits rideStarted to passenger + driver
6
Ride in progress → Driver emits location every 5s → Server emits driverLocationUpdated
7
Driver marks complete → Server emits rideCompleted to passenger + driver (with final fare)
✕
Cancelled at any point → Server emits rideCancelled to passenger + driver

📡 driverLocationUpdated → Ride Room (Passenger + Driver)

Driver ka real-time location update. Driver client har 5 second mein driver:location_update emit karta hai — server ise ride room mein driverLocationUpdated ke naam se broadcast karta hai. Map pe smooth animation ke liye use karo.

Triggered by client emit (Driver App)
socket.emit("driver:location_update", { latitude, longitude, rideId, accuracy, speed })
Driver Client → Server
{
  "latitude": 28.6250,
  "longitude": 77.2100,
  "rideId": "uuid",
  "accuracy": 10,      // meters
  "speed": 25          // km/h
}
Server → Ride Room
{
  "event": "driverLocationUpdated",
  "rideId": "uuid",
  "driverId": "uuid",
  "location": {
    "latitude": 28.6250,
    "longitude": 77.2100
  },
  "accuracy": 10,
  "speed": 25,
  "timestamp": "2026-05-15T10:40:05Z"
}
⚠️ Important: Driver aur Passenger dono ko socket.emit("ride:join", { rideId }) karna zaroori hai tabhi ride room join hoga aur yeh event milega.
FieldTypeRequiredDescription
location.latitudenumberRequiredDriver latitude
location.longitudenumberRequiredDriver longitude
accuracynumberOptionalGPS accuracy in meters
speednumberOptionalSpeed in km/h
Implementation Code
💻 socket.events.js — 6 emit functions
Add in src/infrastructure/websocket/socket.events.js
export const emitRideAccepted = (passengerId, driverUserId, payload) => {
    const data = { event: 'rideAccepted', timestamp: new Date().toISOString(), ...payload };
    emitToPassenger(passengerId, 'rideAccepted', data);
    emitToDriver(driverUserId, 'rideAccepted', data);
};

export const emitDriverArrived = (passengerId, driverUserId, payload) => {
    const data = { event: 'driverArrived', timestamp: new Date().toISOString(), ...payload };
    emitToPassenger(passengerId, 'driverArrived', data);
    emitToDriver(driverUserId, 'driverArrived', data);
};

export const emitRideStarted = (passengerId, driverUserId, payload) => {
    const data = { event: 'rideStarted', timestamp: new Date().toISOString(), ...payload };
    emitToPassenger(passengerId, 'rideStarted', data);
    emitToDriver(driverUserId, 'rideStarted', data);
};

export const emitRideCompleted = (passengerId, driverUserId, payload) => {
    const data = { event: 'rideCompleted', timestamp: new Date().toISOString(), ...payload };
    emitToPassenger(passengerId, 'rideCompleted', data);
    emitToDriver(driverUserId, 'rideCompleted', data);
};

export const emitRideCancelled = (passengerId, driverUserId, payload) => {
    const data = { event: 'rideCancelled', timestamp: new Date().toISOString(), ...payload };
    emitToPassenger(passengerId, 'rideCancelled', data);
    if (driverUserId) emitToDriver(driverUserId, 'rideCancelled', data);
};

export const emitDriverLocationUpdated = (rideId, passengerId, driverUserId, location) => {
    const io = getIO();
    const data = {
        event: 'driverLocationUpdated',
        rideId, driverId: driverUserId, location,
        timestamp: new Date().toISOString()
    };
    io.to(`ride:${rideId}`).emit('driverLocationUpdated', data);
};
