🔐 4. Authentication Flow (Access Token + Auto-Rotation)

📥 Extract Bearer Token from Header
⬇️
🔴 Check Redis Blacklist (revoked?)
⬇️ Not blacklisted
🔑 JWT Verify (HS256, check signature + expiry)
⬇️ Valid
🗄️ Check sessions DB (token exists?)
⬇️ Found
⏳ Token expires in < 6 hours?
YES → Generate new token → Update DB → Set x-new-access-token header
NO → Continue normally
⬇️
✅ Attach user to req → next()
Manual DB delete = Instant 401
Logout = Redis blacklist 24h
Rotation = Transparent to client

🪪 9. KYC Verification Pipeline

6 Required Documents

🪪 AadhaarOCR → Cashfree
📄 PANOCR → Cashfree
🪪 Driving LicenceOCR → Cashfree
🚗 Vehicle RCOCR → Cashfree
🏦 Bank AccountPenny Drop
🤳 SelfieFace Match

Verification Flow

📸 Driver Submits Document (S3 upload)
⬇️
🔍 Cashfree API → OCR / Penny Drop / Face Match
⬇️
📊 Confidence Score (0-100)
≥ 85 → ✅ Auto-Verified | 60-84 → 👀 Manual Review | < 60 → ❌ Auto-Rejected
⬇️
🔎 Cross-Verification: Name match across docs? Duplicate hash?
⬇️
🚩 Fraud Flag
If mismatch found
✅ Approved
All clear
⬇️
🎉 All 6 docs verified → Driver KYC = VERIFIED → Can go online
